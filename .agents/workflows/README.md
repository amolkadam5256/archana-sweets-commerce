# Growthik Media: Standard Operating Procedures (SOPs)

This directory contains standardized, step-by-step workflows for recurring tasks across the Growthik Media project. These ensure consistency and quality regardless of which agent or developer is executing the task.

## Available Workflows

| Workflow | Description |
|----------|-------------|
| [full-project-audit](workflows/full-project-audit.md) | Comprehensive audit of frontend, backend, SEO and analytics. |
| [seo-content-check](workflows/seo-content-check.md) | Quality gate for every new page or blog post. |
| [analytics-event-setup](workflows/analytics-event-setup.md) | Standard process for adding new GTM/dataLayer events. |
| [production-deployment](workflows/production-deployment.md) | Safe deployment checklist for production updates. |
