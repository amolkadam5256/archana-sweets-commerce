# Workflow: SEO Content Quality Check

Execute this before finalizing any new blog post or landing page.

1. **Heading Structure**
   - [ ] Exactly one H1 containing the primary high-intent keyword.
   - [ ] H2s used for main sections; H3s for sub-points. (No skipped levels).

2. **On-Page Metadata**
   - [ ] Title tag: `< 60 chars`, includes keyword + brand.
   - [ ] Meta description: `140-160 chars`, includes CTA and primary keyword.
   - [ ] Canonical tag is present and correct.

3. **AI Search (AEO) Optimization**
   - [ ] "What is [Topic]?" definition in the first 100 words.
   - [ ] At least one structured list (bullet/numbered) for step-by-step extraction.
   - [ ] At least one comparison table or pros/cons list.
   - [ ] Author bio and "Last Updated" date are visible.

4. **Internal Linking**
   - [ ] Link to at least one relevant "Service" page.
   - [ ] Link from a "Pillar" page to this new content.
   - [ ] No broken links within the content area.

5. **Schema Markup**
   - [ ] `BlogPosting` or `Article` schema for blogs.
   - [ ] `FAQPage` schema if specific questions are answered.
   - [ ] `Service` or `LocalBusiness` associations are verified.

6. **Images & Assets**
   - [ ] All images have descriptive Alt text (max 12 words).
   - [ ] Images are appropriately sized/compressed (WebP preferred).
