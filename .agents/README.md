# Growthik Media Agent System

This folder is the reusable operating system for analytics, SEO, GTM, ads tracking, audits, deployment SOPs, AI prompts and future project implementation.

## Core Areas

- `audits/`: project audits and diagnostic reports.
- `reports/`: final summaries, deployment reports and status reports.
- `exports/`: GTM, GA4, Meta and schema exports.
- `templates/`: reusable implementation templates.
- `prompts/`: reusable AI prompts.
- `instructions/`: setup guides and SOPs.
- `skills/`: reusable AI skills.
- `schemas/`: event, dataLayer and naming schemas.
- `checklists/`: QA, launch and production checklists.
- `debugging/`: debugging playbooks.
- `implementations/`: project-specific implementation docs.
- `automation/`: monitoring and validation instructions.
- `analytics/`: analytics architecture and GTM docs.
- `seo/`: technical, on-page, local and schema SEO systems.
- `ads/`: paid media tracking systems.
- `tracking/`: platform-specific tracking references.
- `reusable/`: cross-project systems.
- `ai-system/`: AI agent orchestration and reusable agent workflows.
- `archive/`: deprecated or historical files.
